# Read Parcel Documentation . 

## Key Features

- Dev Build
- Local Server
- HMR (Hot Module Replacement)
- File Watching Algorithm (C++)
- Caching (stores in `.parcel-cache`)
- Image Optimization
- Bundling
- Compression
- Consistent Hashing
- Code Splitting
- Differential Bundling (legacy browser support)
- HTTPS
- Diagnostics
- Error Handling
- Tree Shaking (removes unused code)
